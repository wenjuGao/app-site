### 整数型（int）

正数、负数、0
- 十进制：默认
- 二进制：0b开头
- 八进制：0O开头
- 十六进制：0x开头


### 浮点数（float）

### 布尔（bool）

类似于javascript参数数值运算时(注意大小写，py中大写)：
False => 0
True => 1
